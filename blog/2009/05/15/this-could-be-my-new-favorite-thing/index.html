This Could Be My New Favorite Thing

May 15th, 2009

I was just watching Rob Conery’s screencast on BDD and he showed me a tool that I think might just be my new favorite thing: MSpec/machine.specifications.

It’s a really cool new BDD style executable specification framework which is heavily inspired by RSpec and uses some really neat c# syntax tricks. Not only does it offer a great specification DSL in code, but it, like RSpec, outputs a document that is the specifications document.

Here is a sample snagged out of one of the example projects:

In addition, it has integration with all sorts of great tools. Resharper for instance: it provides a test runner plugin for Resharper 4.1 and 4.5. Here’s a quick screenshot:

And here it is in the code editor window:

It also provides integration with TeamCity, outputting the specifications documentation, with current pass/fail/not-implemented results right there in the specifications.

Go check it out! Watch the screencast and gaze in wonderment!
